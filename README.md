# hold-order
상품 주문의 트래픽에 유연하게 동작하는 프로젝트입니다.  
객체지향 원칙을 잘 지키며 확장과 변화에 유연하게 대체할 수 있도록 개발했습니다.  
테스트를 꼼꼼하게 하였습니다.


## 사용 기술
- JAVA 17
- SpringBoot 3.2.5
- JUnit 5
- MySQL

- swagger
